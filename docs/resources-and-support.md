## Support

1. Review the relevant doc page for your task.
2. Run **Lint Sheets** and check highlighted cells.
3. Check the browser console for detailed errors (View → Developer → JavaScript Console).
4. Contact your project lead or system administrator.
5. Report bugs at the plugin repository.
