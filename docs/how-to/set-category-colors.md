# Set category colors

Category colors come from the **background color** of the Name column in the Categories sheet.

## Steps

1. In **Categories**, select the cells in column A (category names).
2. Click the **Fill color** tool in Google Sheets.
3. Choose a background color for each category.

## Notes

- There is no separate Color column.
- Do not paste hex codes into the **Applies** column.
- Choose distinct, colorblind-friendly colors when possible.
- Bright, high-contrast colors tend to display best.
