# Getting started

## Overview

The CoMapeo Config Spreadsheet Plugin lets you define categories, fields, icons, colors, and translations in Google Sheets, then export a `.comapeocat` file for CoMapeo.

What you can do:

- Define observation categories
- Create custom data fields
- Add icons and colors
- Manage translations
- Export and import configurations

## Prerequisites

- A Google account with access to Google Sheets
- The template spreadsheet:
  https://docs.google.com/spreadsheets/d/1bvtbSijac5SPz-pBbeLBhKby6eDefwweLEBmjAOZnlk/edit?usp=drivesdk
